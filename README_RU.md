# ИИ Бот-Саппорт (Nico Market)

[![English](https://img.shields.io/badge/Language-English-red.svg)](README.md)
[![Russian](https://img.shields.io/badge/Language-Russian-blue.svg)](README_RU.md)

Продвинутый, готовый к production ИИ-ассистент для Telegram. Проект построен на архитектуре RAG (Retrieval-Augmented Generation), использует локальные нейросети через Ollama и векторную базу данных ChromaDB для точного семантического поиска.

## 🚀 Ключевые возможности

- **100% Локальная работа**: Использование Ollama (например, Qwen 2.5 Coder 32b) обеспечивает полную конфиденциальность данных и отсутствие затрат на API.
- **Продвинутая RAG-Архитектура**: 
  - Семантический поиск через `SentenceTransformers` и `ChromaDB`.
  - Калибровка косинусного расстояния (`cosine`) и нормализация векторов для максимальной точности.
  - **Контекстный поиск**: Бот анализирует историю диалога, чтобы понимать уточняющие вопросы пользователя.
- **Защита от галлюцинаций**: Жесткие системные промпты запрещают модели выдумывать несуществующие товары или услуги. Дополнительно работают **Guardrails** (Regex-валидация), которые фильтруют сгенерированные телефоны, email и ссылки, оставляя только официальные контакты.
- **Асинхронность и Скорость**: Написан на `aiogram 3.x` с использованием `asyncio.to_thread`. Тяжелые математические вычисления (создание эмбеддингов) вынесены в отдельные потоки, поэтому бот не "зависает" под нагрузкой.
- **Готовность к Production**: 
  - Глобальный `ErrorHandlingMiddleware` отлавливает ошибки и вежливо извиняется перед пользователем.
  - Фоновый сборщик мусора автоматически очищает устаревшие сессии из оперативной памяти.
  - Полная поддержка Docker и Docker Compose для деплоя в один клик.

## 🛠 Технологический стек

- **Фреймворк**: `aiogram 3.x` (Python 3.10+)
- **Векторная БД**: `ChromaDB`
- **Эмбеддинги**: `sentence-transformers` (`paraphrase-multilingual-MiniLM-L12-v2`)
- **LLM Движок**: `Ollama` 
- **Сеть**: `aiohttp` (Паттерн Singleton-сессии)

## 📦 Быстрый старт

### 1. Требования
- Установленные Docker и Docker Compose.
- Локально установленная и запущенная [Ollama](https://ollama.ai/).

### 2. Настройка
1. Склонируйте репозиторий.
2. Скопируйте `.env.example` в `.env` и вставьте свой токен Telegram-бота:
   ```env
   BOT_TOKEN=ваш_токен_здесь
   OLLAMA_BASE_URL=http://host.docker.internal:11434
   OLLAMA_MODEL=qwen2.5-coder:32b
   ```
3. Положите ваши текстовые файлы с правилами магазина (.txt, .md) в папку `data/knowledge_base/`.

### 3. Запуск
Запустите проект одной командой:
```bash
docker-compose up --build -d
```
При старте бот автоматически проверит связь с Ollama, проиндексирует базу знаний и начнет работу.
